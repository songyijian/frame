矩形：rect

attr

x="20" y="20" rx="20" ry="20" width="250" height="100"

  • x , y - 坐标右上角
  • rx，ry - 圆角（rx 横向曲线，ry纵向曲线）
  • width，height - 宽，高

style 属性

style="fill:red;stroke:blue;stroke-width:5;fill-opacity:0.5;stroke-opacity:0.9; opacity:0.5"

  • fill:red; - 填充
  • fill-opacity:0.5; - 填充透明度
  • stroke:red; - 边框颜色
  • stroke-width:5; - 边框宽度
  • stroke-opacity:0.9; - 边框透明度
  • opacity:0.5 - 整个svg透明度

圆形：circle

cx="100" cy="50" r="40" stroke="black"

  • cx,cy - 原点坐标 - 圆心坐标（cx 和 cy 属性定义圆点的 x 和 y 坐标。如果省略 cx 和 cy，圆的中心会被设置为 (0, 0)）
  • r="40" - 圆半径

椭圆： ellipse

  • cx,cy - 原点坐标
  • rx ，ry - 半径

线： line

x1="0" y1="0" x2="300" y2="300"

  • x1，y1 - 起点坐标
  • x2，y2 - 终点坐标

多边形:polygon

points="220,100 300,210 170,250"

  • points="x,y x,y x,y" - 每个点的坐标

折线： polyline

points="0,0 0,20 20,20 20,40 40,40 40,60"

  • points="x,y x,y x,y" - 每个点的坐标

路径： path

  • M = moveto
  • L = lineto
  • H = horizontal lineto
  • V = vertical lineto
  • C = curveto
  • S = smooth curveto
  • Q = quadratic Belzier curve
  • T = smooth quadratic Belzier curveto
  • A = elliptical Arc
  • Z = closepath
